# ⚙️ Installation

### 1️⃣ Clone the Repository

``` bash
git clone https://github.com/VinayagamRaja/developer-portfolio-react.git
```

### 2️⃣ Navigate to Project Folder

``` bash
cd developer-portfolio-react
```

### 3️⃣ Install Dependencies

``` bash
npm install
```

### 4️⃣ Run Development Server

``` bash
npm start
```

Application runs at:\
http://localhost:3000
